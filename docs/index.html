#Important Concepts

Game Design Concepts

Concept Description Example
Mechanics Rules and systems that define how the game is played. Jumping, Shooting
Dynamics Player interactions that arise from the mechanics. Strategy, Puzzle-solving
Aesthetics The visual and audio elements that define the game's style. 8-bit graphics, Chiptune music
